A simple developer toolbar for react based apps.

## Installation

```bash
npm install devbar
```

## Usage

```jsx
import React from 'react'
import DevBar from 'devbar'

const App = () => {
  return (
    <div>
      <h1>Hello World</h1>
      <DevBar />
    </div>
  )
}
```
